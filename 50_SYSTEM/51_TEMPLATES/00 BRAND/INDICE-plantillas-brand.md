## Cómo Usar

1. **Copiar la plantilla principal** `plantilla-brand-identity.md`
2. **Renombrar** `[TuProyecto]-brand-identity.md`
3. **Reemplazar** `{{title}}` con el nombre del proyecto
4. **Completar** cada sección siguiendo las sub-plantillas

### Flujo de Trabajo

```
1. Estrategia Base (A1-A5)
 A1: Público Objetivo
 A2: Competidores
 A3: Propuesta de Valor
 A4: Valores de Marca
 A5: Personalidad y Voz
 
2. Identidad de Marca (B1-B7)
 B1: Naming y Dominio
 B2: Tipografía
 B3: Paleta de Colores
 B4: Logotipo
 B5: Vectorización
 B6: Iconos
 B7: Mockups
```
