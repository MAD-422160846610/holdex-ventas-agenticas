# Doble Corchete (Wikilinks en Obsidian)

Esta nota explica el sistema de [[wikilinks|enlaces]] en Obsidian.

## ¿Qué son los dobles corchetes?
Los enlaces en Obsidian usan el formato `[[nombre-de-nota]]` para conectar notas.

## Para qué sirven
- Conectar notas entre sí
- Crear una red de conocimiento
- Navegar rápidamente

## Uso básico
```
[[Nombre de la Nota]]
[[Nombre de la Nota#Encabezado]]
[[Nombre de la Nota|Alias]]
```
