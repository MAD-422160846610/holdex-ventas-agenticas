# B4 - Logotipo e Isotipo: Holdex Sales Division

**Proyecto:** Piso 01 - Marketing y Ventas
**Sección:** B) Identidad Visual

---

## Concepto Base
El logotipo de Holdex Sales no es un logo tradicional corporativo. Debe sentirse como el prompt de una terminal de comandos que tiene el control de una máquina poderosa. Es austero, técnico y brutalista.

## Isotipo (El Símbolo)
- **Concepto:** El "Bracket Cursor".
- **Visual:** Un corchete de apertura seguido de un cursor parpadeante: `[ _`
- **Significado:** Representa el inicio de un comando, la orquestación (el corchete contiene) y la acción inminente (el cursor espera).
- **Color:** Cyber Green (#00FF00) para el estado activo, Holdex Gold (#C9A227) para servicios premium/capacitación.

## Logotipo (La Tipografía)
- **Concepto:** Texto crudo, sin adornos.
- **Fuente:** JetBrains Mono ExtraBold (800).
- **Estructura:** `HOLDEX_SALES` (siempre en mayúsculas, siempre con guion bajo separando palabras).
- **Alineación:** Monolineal.

## Imagotipo (Logo + Símbolo)
**Disposición Horizontal (Principal):**
`[ _ HOLDEX_SALES ]`
- El texto está contenido dentro de corchetes, imitando un array o un tag de sistema.

**Disposición Vertical (Secundaria/Móvil):**
```
 [ _ ]
HOLDEX_
SALES_
```

## Reglas de Aplicación
1. **Zonas de Seguridad:** El logo debe tener un espacio de padding igual a la altura de los corchetes en todas las direcciones.
2. **Uso sobre fondos:** SIEMPRE sobre fondo Void Black (#000000) o Deep Black (#080808). NUNCA sobre blanco.
3. **Versión Monocromática:** Para impresos o marcas de agua, usar Terminal White (#F0F0F0) o Gray 400 (#9CA3AF). NUNCA usar el logo en negro (se perdería en el fondo de la marca).
