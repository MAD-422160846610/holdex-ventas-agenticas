# A5 - Personalidad de Marca: Sabueso

**Proyecto:** Piso 01 - Marketing y Ventas
**Seccion:** A) Estrategia Base

---

## Arquetipo de Marca: EL ARQUITECTO + EL REBELDE

Sabueso combina la precision del **Arquitecto** (estructura, sistema, confiabilidad) con la irreverencia del **Rebelde** (desafía el status quo, rechaza la mediocridad, rompe paradigmas).

---

## Personalidad en 5 Dimensiones

### 1. Voz
- **Directa:** Sin rodeos. Decimos las cosas como son.
- **Técnica pero accesible:** Usamos terminología de sistemas pero la explicamos con analogías de construccion.
- **Con autoridad:** No pedimos permiso. Presentamos soluciones.

### 2. Tono
- **Confiante sin ser arrogante:** "Esto funciona porque lo usamos todos los días."
- **Urgente pero no desesperado:** "El 2026 ya llegó. ¿Todavía vendés como en 2015?"
- **Cálido en la capacitacion:** Cuando enseñamos, somos pacientes y apasionados.

### 3. Estetica Visual
- **Cyber-Brutalismo:** Bordes gruesos, sombras sólidas, tipografía monoespaciada.
- **Paleta:** Negro absoluto + Verde neón (#00FF00) + Oro corporativo (#C9A227).
- **Metáfora visual:** Terminal de comando, dashboard de control, sala de operaciones.

### 4. Lenguaje Clave (Lexico de Marca)
| En vez de... | Decimos... |
|-------------|-----------|
| Chatbot | Agente Autónomo |
| CRM | Vault de Conocimiento |
| Vendedor | Board Operator |
| Software | Arnés (The Harness) |
| Automatización | Orquestación |
| Datos | Soberanía del dato |
| Dashboard | Centro de Mando |
| Plan | Piso (Floor) |

### 5. Comportamiento en Canales
| Canal | Estilo |
|-------|--------|
| **LinkedIn** | Profesional, datos duros, micro-casos de éxito |
| **WhatsApp** | Directo, breve, con "comandos" estilo terminal |
| **Landing Page** | Cinematográfico, brutalista, impactante |
| **Email** | Corto, con un CTA claro, sin imágenes innecesarias |
| **Capacitación** | Cálido, didáctico, con analogías de construcción |

## El "Test de Personalidad"

Si Sabueso fuera una persona:
- **Película favorita:** The Matrix (1999)
- **Herramienta favorita:** Un terminal Linux con Vim
- **Bebida:** Café negro, sin azúcar
- **Música:** Lo-fi beats o synthwave mientras trabaja
- **Frase que repetiría:** "Orquestá, no programés"
