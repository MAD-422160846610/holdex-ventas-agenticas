# B3 - Paleta de Colores: Holdex Sales Division

**Proyecto:** Piso 01 - Marketing y Ventas
**Seccion:** B) Identidad Visual

---

## Paleta Principal

| Nombre | Hex | RGB | Uso |
|--------|-----|-----|-----|
| **Void Black** | #000000 | 0, 0, 0 | Fondo principal |
| **Deep Black** | #080808 | 8, 8, 8 | Fondo de tarjetas |
| **Cyber Green** | #00FF00 | 0, 255, 0 | Acentos, CTAs, elementos interactivos |
| **Holdex Gold** | #C9A227 | 201, 162, 39 | Acentos secundarios, premium tier |
| **Terminal White** | #F0F0F0 | 240, 240, 240 | Texto principal |

## Paleta Secundaria

| Nombre | Hex | RGB | Uso |
|--------|-----|-----|-----|
| **Alert Red** | #DC2626 | 220, 38, 38 | Errores, alertas, seccion "Problema" |
| **Gray 400** | #9CA3AF | 156, 163, 175 | Texto secundario, descripciones |
| **Gray 700** | #374151 | 55, 65, 81 | Bordes sutiles, separadores |
| **Border Dark** | #111111 | 17, 17, 17 | Bordes de tarjetas en reposo |
| **Border Active** | #222222 | 34, 34, 34 | Bordes interactivos |

## Gradientes

| Nombre | Definicion | Uso |
|--------|-----------|-----|
| **Text Gradient** | #00FF00 → #C9A227 | Texto destacado (H1, taglines) |
| **Glow Green** | rgba(0,255,0,0.4) drop-shadow | Efecto de brillo en ASCII y elementos key |
| **CTA Fade** | rgba(0,255,0,0.05) → transparent | Fondos sutiles de secciones de accion |

## Reglas de Contraste

- **Fondo siempre negro:** Nunca usar fondos claros. La marca vive en la oscuridad.
- **Texto sobre negro:** Siempre Terminal White (#F0F0F0) o Gray 400.
- **Green solo para acciones:** No usar Cyber Green en texto de lectura. Es para CTAs, bordes activos y elementos interactivos.
- **Gold para premium:** Reservar Holdex Gold para elementos que comuniquen valor, exclusividad o el tier de capacitacion.
