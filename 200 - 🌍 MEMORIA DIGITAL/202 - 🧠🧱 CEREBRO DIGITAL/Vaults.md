#🗄️ Vaults en Obsidian


#¿Qué es un Vault?

Un vault (bóveda) es tu colección personal de notas. Es tu segundo cerebro digital.


#Estructura de un Vault

```
Mi Vault/
├── .obsidian/          # Configuración de Obsidian
├── Attachments/        # Imágenes y archivos
├── Templates/          # Plantillas de notas
├── Projects/           # Notas de proyectos
├── Resources/          # Referencia
└── Daily/              # Notas diarias
```


#Principios de organización


#1. Un solo vault para todo
Evita fragmentar tu conocimiento en múltiples vaults.


#2. Estructura simple
Carpetas para áreas grandes, tags para detalles.


#3. Revisión periódica
Limpiar y reorganizar regularmente.


#Configuración recomendada


#Plugins esenciales
- Dataview: Consultas de notas
- Templater: Creación de notas
- Daily Notes: Notas diarias
- Tag Wrangler: Gestión de tags


#Configuración de sync
- Git: Control de versiones
- Sync de Obsidian: Entre dispositivos
- Backup automático: Seguridad


#Buenas prácticas


#Nombrar archivos
- Usar guiones: `mi-nota.md`
- No espacios: `mi-nota` no `mi nota`
- Descriptivo: `Presupuesto-ITB-2026.md`


#Enlazar
- Conectar ideas relacionadas
- Revisar enlaces rotos
- Mantener bidireccionalidad


#Mantener
- Actualizar notas obsoletas
- Archivar lo que ya no sirve
- Celebrar el crecimiento
